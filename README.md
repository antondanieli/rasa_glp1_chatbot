# GLP-1 Support Chatbot

This is a simple support chatbot built with **Rasa** for a GLP-1 product website.  
It helps answer common customer questions, guide users through product information, and handle basic support tasks.

## How to run

### Train the model
rasa train

### Start the action server (if using custom actions)
rasa run actions

### Start the chatbot
rasa shell

## Project overview
- Built with Rasa Open Source
- Includes intents, stories, rules, and responses
- Easy to customize for new product flows or FAQs

## Purpose
Provide automated support and guidance for users interested in GLP-1 products.
